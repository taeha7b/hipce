## 적용 기술 및 구현된 기능
### 적용 기술
* Python, Django web framework
* Bcrypt
* JWT
* MySQL
* AWS EC2, RDS

### 구현된 기능
* Bcrypt를 활용한 비밀번호 암호화
* JWT를 활용한 로그인 토큰 발행
* 회원가입시 정규식을 사용한 회원정보 유효성 검사
* 리뷰 작성
* 리뷰에 댓글 작성
